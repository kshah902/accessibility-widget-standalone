Accessibility Widget Test Page

This is a test page to verify the standalone accessibility widget works correctly. The widget should appear as a blue circular button in the bottom-right corner of the screen.

Test Features

Click the accessibility button to open the panel and test:

  • Text Size - Try changing the text size using the buttons
  • High Contrast - Toggle high contrast mode
  • Dyslexia Font - Enable the OpenDyslexic font
  • Color Blindness Filters - Test different color blindness simulations
  • Reading Guide - Enable the line that follows your cursor

Keyboard Shortcuts

Try these keyboard shortcuts:

  • Option/Alt + A - Toggle the accessibility panel
  • Option/Alt + C - Toggle high contrast
  • Option/Alt + M - Toggle reduce motion
  • Option/Alt + R - Reset all settings
